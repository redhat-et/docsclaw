## Finance analyst — fan-out demo

Five contracts (DOC-CTR-V001..V005), each with three monthly
invoices. Five parallel finance-analyst agents, one contract each.

**Result (DOC-CTR-V003 — BandwidthPro, representative):**
- Time: ~90 seconds (4 tool calls: 1 contract + 3 invoices)
- Found: $15,000 uncontracted "Consulting Services" charge on
  January invoice, storage growth trend (9.6% MoM), bandwidth
  fluctuation pattern
- All contracted rates verified correct across 3 months
- Output: structured anomaly report with rate deviations,
  duplicate charges, uncontracted items, unusual patterns

**Key demo point:** five identical 11 MiB pods running in
parallel, each analyzing a different vendor — total cluster
footprint under 60 MiB for the entire fan-out.

## Security analyst — multi-document cross-reference

Single agent, two documents: vulnerability scan (50 CVEs) and
infrastructure asset inventory (8 hosts, 3 teams, 3 SLA tiers).

**Result:**
- Time: ~52 seconds (2 LLM calls, 2 tool calls, 7,767 total
  tokens)
- Identified 5 hosts in SLA breach (32 findings), 2 hosts
  imminent (15 findings)
- Worst breach: cache-prod-01 at 30 days overdue with 10
  findings
- Produced team-grouped remediation plan with severity
  breakdown and 9 prioritized action items
- All 8 hosts matched against inventory — no orphaned findings

**Key demo point:** the agent correlated two datasets and applied
business logic (SLA deadlines, severity, team ownership) to
produce a prioritized remediation plan — work that takes a human
analyst 30-60 minutes.

**Efficiency highlight:** 50 CVEs processed in 2 LLM calls and
~3,700 input tokens. The compact table format keeps token usage
low despite the data volume.
